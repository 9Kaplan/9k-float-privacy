What 9K Float does

9K Float lets you select a visible area of a supported webpage, save it locally, and open it in Chrome Picture-in-Picture while the source tab remains open.

How to test the extension

  1. Open a normal webpage, such as an article, dashboard, or reference page.
  2. Click the 9K Float toolbar icon.
  3. Click “Create floating area”.
  4. Select a visible rectangular area on the page.
  5. Save the area.
  6. Open the saved area in Picture-in-Picture.
  7. Use the library to reopen, rename, adjust, or delete saved areas.

Common limitations

  • Chrome can show only one Picture-in-Picture window at a time.
  • The source tab must stay open for the floating area to keep updating.
  • Some pages, including protected Chrome pages, Chrome Web Store pages, browser pages, and restricted sites, may not be supported.
  • Classic Chrome Picture-in-Picture does not provide full extension control over the operating system window size.

Troubleshooting

  • If a saved area does not open, make sure the original source tab is still open.
  • If Chrome says PiP is busy, close the current floating window before opening another saved area.
  • If a page is unsupported, try selecting an area from a normal webpage instead of a protected browser or store page.
  • If a preview thumbnail is missing, the area may have been saved from a restricted page or a capture may have failed; the saved area can still use a placeholder.
