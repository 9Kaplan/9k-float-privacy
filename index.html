What 9K Float does

9K Float lets you save a visible area of a supported webpage and float it in Chrome Picture-in-Picture while the source tab remains open.

Common limitations

  • Chrome can show only one Picture-in-Picture window at a time.
  • The source tab must stay open for the floating area to work.
  • Some pages, including protected Chrome pages and Chrome Web Store pages, are unsupported.
